**正则序求值**  
正则序求值是完全展开而后归约的求值方式。

**应用序求值**   
应用序求值是先求值参数而后应用的方式。  

sicp 的1.5 题

<pre>
(define (p) (p))
(define (test x y)
    (if (= x 0)
         0
        y))

(test 0 (p))
</pre>

1.如果解释器采用正则序求值的话，完全展开的话是
<pre>
(test 0 (p)
	(if (= 0 0 )
		0
		y))
</pre>
解释器输出的是0

2.如果解释器采用的是应用序求值的话。解释器会先求值参数，因为参数中(p)  
所以会一直调用自己。

lisp采用应用序求值，这样做可以避免对于表达式的重复求值。
