git 简史

history

linusTorvald

Git 是 Linus Torvalds 为了帮助管理 Linux 内核开发而开发的一个开放源码的版本控制软件。

特点:

速度
简单的设计
对非线性开发模式的强力支持（允许上千个并行开发的分支）
完全分布式
有能力高效管理类似 Linux 内核一样的超大规模项目（速度和数据量）

|-- HEAD         # 这个git项目当前处在哪个分支里
|-- config       # 项目的配置信息，git config命令会改动它
|-- description  # 项目的描述信息

|-- logs/        # 各个refs的历史信息
|-- objects/     # Git本地仓库的所有对象 (commits, trees, blobs, tags)
`-- refs/        # 标识你项目里的每个分支指向了哪个提交(commit)。

在github上建立个人主页

创建一个用户的页面很简单，假设你的用户名为foo

1. 新建一个仓库(repository)，名称填"foo.github.com"，创建
2. 在本地新建一个文件夹foo,在foo里面新建一个index.html页面
3. `git init` 初始化，`git add .` 把index.html加入到仓库中，`git commit` 提交
4. 添加github的分支
5. 提交到github分支 `git push origin master`
浏览foo.github.com，就可以看到刚刚创建的index.html文件了。
在你的github项目下增加CNAME文件，内容为你的域名
